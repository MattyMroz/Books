# Tworzenie Audiobooków
---

## Istnieją 3 możliwości które:
- Samodzielnie odkryłem, przetestowałem i stosuje
- Są darmowe
- Mają dobrą jakość — głos syntezatora
---

#### Tekst -> VSC -> Word -> VSC -> GitHyb Pages -> WWW
- [VSC](https://code.visualstudio.com/) -> index.html -> w znacznikach \<p> Wklej tekst \</p> -> Shift + Alt + F (Formatuj dokument)
- Tekst -> Word -> Ctrl + H -> Więcej -> Specjalne -> (Znajdź) Znak akapitu -> (Zamień na) \<br> -> Zamień wszystko
- [VSC](https://code.visualstudio.com/) -> Zbuduj prostą strone internetową -> Powinna zawierać linki do podstron z rozdziałami - lepsze pobieranie w przeglądarce
- W index.html linki w takiej formie:<br>
```html
 <li>
	<a href="/Nazwa_Repozytorium/index2.html">1.</a>
 </li>
```
- Utwórz repozytorium publiczne na [GitHub](https://github.com/) -> Repozytorium -> Settings -> Pages -> None ---> main -> /(root) ---> /docs -> Save -> Kopiuj link do strony -> POZIOM ŁATWY — czytanie z poziomu przeglądarki
